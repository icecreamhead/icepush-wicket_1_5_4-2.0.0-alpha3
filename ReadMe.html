Welcome to ICEpush 2.0.0-Alpha3! Wicket Integration Bundle

This bundle contains software source-code and executables for ICEpush 2.0.0-Alpha3 and the Wicket integration and samples.

All documentation related to ICEpush and the use of this bundle can be found in the ICEpush Wiki .

  • samples:
    • pushpanel: A basic test of ICEpush with the Wicket integration and PushPanel component
    • icechat: A chat demo demonstrating the ICEpush and the Wicket integration

To Build

To build this bundle, run the default target in the root ant script 'build.xml': 

ant
which will build the libraries and all samples.
